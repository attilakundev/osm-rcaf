This tool was made by Attila Kun by using/reconstructing the base-logic of existing tools(JOSM and OSM Relation Analyzer) and graph theory.

The tool has its roots at OSM Wiki/Hungary/Szerszámzat#Reláció szakadás ellenőrző.

The project's repo is available at: https://github.com/attilakundev/osm-rcaf

How to use it?

To use it, you need to supply a relation ID (eg. 67157), or you upload a file where you supply the relation ID to be searched for.

When analyzing is done, then you can self evaluate the result and fix the relation with this tool. Right now it only supports resupplying XML file, but the starting way can be chosen from a list.

When the correction is done, then the xml file gets downloaded on your PC. If you want it, you can head to the compare page (see the navbar) to let your original xml compared to your corrected one, and then see the result.
